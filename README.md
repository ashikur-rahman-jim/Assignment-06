# Digital Tools Buying Website
### Digital Tools Buying Website is a modern web application where users can explore and purchase various digital tools such as AI software, templates, and productivity resources.

## Technologies Used
### React.js
### Tailwind CSS
### DaisyUI
### Vite
### Lucide React Icons

## Features
### Product Browsing System : Users can explore different digital tools in an organized and user-friendly interface.
### Add to Cart Functionality : Users can add products to the cart and manage selected items dynamically.
### Dynamic UI with Tabs : Seamless switching between Products and Cart without page reload using React state.
